A

极简 HUD

只有准星 + 左下部位剪影 + 右下弹药/速度。沉浸感强，队友、小地图、比分按 Tab/M 呼出。

B

信息丰富 HUD

常驻小地图 + 队伍名单 + 计时比分 + 剪影+部位血条 + 弹药/装填/速度。一眼看懂战局，但画面更"忙"。

C

混合：默认 A、按 Tab 临时显示 B

平时沉浸，需要全局信息时按一下 Tab 呼出小地图/队友/部位血条面板，松开即收回。
